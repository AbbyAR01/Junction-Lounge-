A modern, fully-responsive full-stack restaurant website built for **Junction Lounge**. Features live menu, online ordering, Google Maps integration, and mobile-first design.

## 🚀 **Features**
- ✅ **Live Menu API** - Dynamic menu with categories (Cocktails, Appetizers, Mains, Desserts)
- ✅ **Online Ordering** - Working order form with API submission  
- ✅ **Fully Responsive** - Perfect on mobile, tablet, desktop
- ✅ **Google Maps** - Embedded location from your link
- ✅ **Smooth Animations** - Modern hover effects and transitions
- ✅ **Fast Loading** - Optimized images and CDN delivery
- ✅ **SEO Ready** - Meta tags and semantic HTML
- ✅ **PWA Ready** - Installable on mobile home screen
